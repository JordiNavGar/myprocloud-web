FAQ

Frequently Asked Questions

Can A3 be migrated to the cloud?

Yes. We execute a full migration of your local A3 directory structures and databases to our cloud servers, maintaining all paths and setup properties.

How do advisors access their A3 tools?

Users log in via secure Remote Desktop Protocol (RDP) sessions over a corporate virtual private network (VPN), giving them a familiar workspace.

What happens with A3 backups?

We run automated offsite backups daily using Veeam Backup. Backups are stored in dedicated encrypted environments to prevent data loss.
