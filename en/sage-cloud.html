FAQ

Frequently Asked Questions

Can I use Sage from outside the office?

Yes. The software and database run in our cloud server environment, allowing secure access via VPN credentials from anywhere with internet connectivity.

What happens to my current Sage license?

You keep your existing Sage license. We manage the software migration and set up databases in the cloud without requiring new ERP licenses.

How are Sage backups handled?

We run daily full backups encrypted and stored off-site via Veeam Backup, ensuring quick recovery if administrative issues arise.
